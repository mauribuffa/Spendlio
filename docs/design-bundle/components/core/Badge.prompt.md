Small status or label pill — settlement state ("Settled", "You owe"), category tags, counts.

```jsx
<Badge tone="positive" dot>Settled</Badge>
<Badge tone="negative">You owe $24.50</Badge>
<Badge tone="brand">Recurring</Badge>
```

Tones: `neutral | brand | positive | negative | warning | info | solid | outline`. Set `dot` for a status dot, `size="sm"` for dense rows.
