Circular icon-only button — for toolbars, card headers, and list-row affordances. Always pass an accessible `label`.

```jsx
<IconButton icon={<MoreIcon/>} label="More options" />
<IconButton icon={<PlusIcon/>} label="Add" variant="brand" />
```

Variants: `ghost` (default, transparent until hover), `solid` (white chip with border), `brand` (green fill). Sizes `sm | md | lg`.
