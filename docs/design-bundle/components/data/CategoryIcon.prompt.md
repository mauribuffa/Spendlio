Category glyph in a soft tinted circle — the Lucide icon and `--cat-*` color for each spend category. Pairs with `TransactionRow`.

```jsx
<CategoryIcon category="groceries" />
<CategoryIcon category="dining" size="lg" square />
```

Categories: groceries, dining, transport, housing, utilities, shopping, health, entertainment, travel, subscriptions, income, transfer (exported as `CATEGORIES`). Call `lucide.createIcons()` after mount so the icon renders.
